# 💊 PharmaSys

Sistema desenvolvido em Python para organização e gerenciamento de documentos farmacêuticos.

## 📋 Sobre o Projeto

O PharmaSys foi criado para agilizar o processo de organização de receitas, notas e documentos em drogarias e farmácias, reduzindo o tempo gasto com tarefas manuais e minimizando erros de arquivamento.

## 🚀 Funcionalidades

- 📁 Criação automática de múltiplas pastas
- 📄 Organização de receitas e notas fiscais
- 🔍 Pesquisa rápida de pastas
- 🖼️ Visualização prévia de imagens antes da movimentação
- ↩️ Desfazer última movimentação realizada
- 📌 Atualização automática da lista de pastas
- 🎨 Interface gráfica moderna desenvolvida com Tkinter
- 📦 Instalador para Windows

## 🛠️ Tecnologias Utilizadas

- Python 3
- Tkinter
- Pillow (PIL)
- PyInstaller
- Inno Setup

## 📦 Instalação

### Utilizando o instalador

1. Baixe o arquivo de instalação.
2. Execute `PharmaSys_Setup.exe`.
3. Siga as instruções do assistente de instalação.
4. Abra o PharmaSys pelo Menu Iniciar ou Área de Trabalho.

### Executando pelo código-fonte

```bash
pip install pillow
python PharmaSys.py
```
